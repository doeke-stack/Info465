Git Assignment 1 Materials

Files included:
1. integer_stats.js
   - JavaScript program that reads integers from the user.
   - Stores valid integers in an array.
   - Calculates and displays mean and median.
   - Handles invalid input such as words, decimals, and blank entries.
   - Allows the user to type q to quit.

How to run the program:
1. Open Command Prompt, PowerShell, or Terminal.
2. Go to the folder where this file is saved.
3. Run:
   node integer_stats.js

Example test:
Enter 5
Enter 3
Enter 9
Enter hello
Enter 4
Enter q

Expected result:
- The program rejects hello.
- The program calculates the mean and median for [5, 3, 9, 4].
- Mean should be 5.25.
- Median should be 4.5.

Suggested Git commit message:
Add JavaScript mean and median calculator
